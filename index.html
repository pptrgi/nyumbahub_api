NyumbaHub API

Welcome to NyumbaHub vacant houses API

Here's what you need to know:

  • Public Endpoints has the route endpoints that are accessible by all users.
  • Routes that are not directly available to the public, or those on Protected Endpoints, either require signing in first or can only be accessed by special/privileged users.
  • To access majority of routes you'll need to sign in. If you haven't registered already, register first.
  • Click on a route to visit it, looks like so, /route/endpoint
  • For POST in user register and login and other HTTP methods including PUT and DEL, you will need Postman or Thunder Client etc
