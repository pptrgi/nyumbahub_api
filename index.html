NyumbaHub API

Welcome to NyumbaHub vacant houses API

Here's what you need to know:

  • The route endpoints that are accessible by all users are those under Public Endpoints, and they don't require signing in.
  • To access majority of routes you'll need to sign in, register first if you haven't already.
  • Routes that are not directly available to the public, or those on Other Endpoints, either require signing in first or can only be accessed by special/priviledged users.
